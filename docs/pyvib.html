localize(option: Optional[str] = 'Pipek-Mezey', window: Optional[float] = 400)[source]

Localize vibrational modes.

Vibration mode localization with local_cls module.

  • Pipek-Mezey metric

\[\begin{split}\xi_{\mathrm{at}}\left(\widetilde{\boldsymbol{Q}}^{\mathrm{sub}}\right)=\sum_{p=1}^{k} \sum_{i=1}^{n}\left(\tilde{C}_{i p}^{\mathrm{sub}}\right)^{2} \\ \tilde{C}_{i p}^{\mathrm{sub}}=\sum_{\alpha=x, y, z}\left(\tilde{Q}_{i \alpha, p}^{\mathrm{sub}}\right)^{2}\end{split}\]

  • Boys metric

\[\begin{split}\xi_{\mathrm{dist}}\left(\widetilde{\boldsymbol{Q}}^{\mathrm{sub}}\right)=\sum_{p=1}^{k}\left(\boldsymbol{R}_{p}^{\text {center }}\right)^{2} \\ \boldsymbol{R}_{p}^{\text {center }}=\sum_{i=1}^{n} \tilde{C}_{i p}^{\mathrm{sub}} \boldsymbol{R}_{i}\end{split}\]
Parameters

  • option (str) – The metric of localization. Boys or Pipek-Mezey.

  • window (float) – window frequency in cm-1.

Examples

>>> sim = Vibration(geom, freq, disp)
>>> sim.localize()
